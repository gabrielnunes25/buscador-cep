# Buscador de CEP

Saiba os detalhes dos CEPs brasileitos.

## Ferramenta

-   [React](https://reactjs.org/)
-   [Axios](https://axios-http.com/)
-   [React Icons](https://react-icons.github.io/react-icons/)
-   [API de CEP Nacional](https://viacep.com.br/)
